# Maintenance Scripts\n\nEsta carpeta contiene scripts temporales/de apoyo para parches manuales y exploracion.\n\nNo forman parte del runtime principal de la app.\n\n## Contenido\n\n- `expand_*.py`: parches de traduccion/campos en `profiler/app.py`.\n- `get_*.py`: inspeccion puntual del codigo fuente.\n- `rewrite*.py`, `write_gui.py`: generadores/reescrituras de snapshots antiguas.\n\n## Uso recomendado\n\nEjecutarlos solo cuando estes haciendo mantenimiento puntual y con backup previo.\n
